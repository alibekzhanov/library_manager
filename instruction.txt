Необходимо создать сайт для библиотекаря, с помощью которого он может вести
учет книг. Сайт должен состоять из 4 разделов (пунктов меню): Books, Visitors, Cards, Statistics.

1. Вкладка с книгами.
На этой странице есть карточки книг. Данные для книг взять из файла data.js c
помощью модулей. Нужные данные лежат в массиве books:
books: [
	{ id: 1, name: 'Аэропорт', author: 'Артур Хейли', year: 1987, count: 4},
	{ id: 2, name: 'Властелин колец', author: 'Джорд Толкин', year: 1954, count: 2},
	{ id: 3, name: 'Отель', author: 'Артур Хейли', year: 1988, count: 2},
	{ id: 4, name: 'Будущее', author: 'Дмитрий Глуховский', year: 2007, count: 13),
	{ id: 5, name: 'Метро 2033', author: 'Дмитрий Глуховский', year: 2011, count: 5}
],

Каждая карточка книги содержит в себе: название книги, автора, год книги,
количество данной книги в библиотеке. А также две кнопки для удаления книги
или изменения её. Каждая книга уникальна, то есть уникальный id. Этим id нужно пометить саму
карточку, чтобы знать какие данные менять или удалять. 

Над карточками книг добавить кнопку для добавления новой книги. При
добавлении новой книги открывается модальное окно, где заполняются данные
о новой книге (имя книги, имя автора, год книги и количество книги, а также
даётся уникальный id для книги, например следующий номер id после последней
книги). Важно! При добавлении или изменении книги необходимо чтобы поля
были не пустыми, а также в поле год должны быть только положительные
цифры.
Подсказка! В качестве плагина для модального окна можно использовать
Magnific Popup
Также пример использования модального окна есть в папке MagnifiPopup.
Можно также изменить текущую книгу (открывается модальное окно с уже
заполненными полями этой книги – данные взять по id карточки). При
сохранении данных, данные карточки меняются. Можно также удалить
существующую книгу (открывается модальное окно, которое спрашивает вы
точно хотите удалить книгу или нет).

2. Вкладка с посетителями.
На этой странице есть список всех посетителей. Данные для книг взять из файла
data.js c помощью модулей. Нужные данные лежат в массиве users:
users: [
	{ id: 1, name: 'Максим', phone: '+856734435'},
	{ id: 2, name: 'Андрей', phone: '+43543534545'},
	{ id: 3, имя: 'Иван', телефон: '+435435435'},
]
Каждая карточка посетителя содержит в себе: имя посетителя, его телефон. А
также две кнопки для изменения и удаления карточки.
Каждый посетитель уникален. У каждого свой id. Этим id нужно пометить саму
карточку, чтобы знать какие данные менять или удалять.
Сверху посетителей есть кнопка для добавления нового посетителя. При
добавлении нового посетителя открывается модальное окно, где заполняются
данные о новом посетителе (имя, телефон, а также даётся уникальный id для
книги, например следующий номер id после последнего посетителя).
Можно также изменить текущего посетителя (открывается модальное окно с уже
заполненными полями – данные взять по id карточки). При сохранении данных,
данные карточки меняются. Можно также удалить существующего посетителя (открывается модальное окно,
которое спрашивает вы точно хотите удалить посетителя или нет).
Важно! При добавлении или изменении посетителя необходимо чтобы поля
были не пустыми и в поле телефон должны быть только цифры.

3. Вкладка с карточками.
Карточка – это одна взятая книга одним посетителем, с датой выдачи и датой
возврата. Дата возврата появляется только тогда, когда вернули книгу.
Изначально эта страница (вкладка пустая). На ней есть кнопка добавить
карточку. При добавлении карточки открывается модальное окно, где нужно
выбрать книгу через select, выбрать посетителя через select, а также выбрать дату
отдачи книги. Данные для select книг и посетителей взять из файла data.js
(написать функции для заполнения select из объекта data). После добавления карточки на этой вкладке появляется карточка примерно следующего
вида (Имя: Alex, Книга: Аэропорт, Дата выдачи: 12.11.2022, Дата отдачи: 15.11.2022).
Каждая карточка уникальна, имеет уникальный id. При выдаче книги количество
ее экземпляров в библиотеке должно уменьшаться на 1. Соответственно, при
возврате – увеличиваться на 1, и карточка после возврата удаляется.
При создании карточки в выпадающем списке с книгами должны отображаться
только те книги, которые есть в библиотеке на данный момент (то есть
количество книг, которых больше 0).

4. Вкладка с популярными книгами.
Топ 5 самых популярных книг. Составить список на ваш выбор. И выводить в
виде таблицы эти книги. Колонки в таблице (id, name, author, year, count)

Подсказка! Хранить данные нужно в localStorage.
Подключение файла data.js через модули:
<script type="module" src="./data.js"></script>
<script type="module" src="./script.js"></script>

Использование полученных данных в вашем файле script.js:
import {data} from "./data.js";
